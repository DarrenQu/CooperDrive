# CooperDrive GitHub Pages Project Page

This is a ready-to-publish academic project page for:

**CooperDrive: Enhancing Driving Decisions Through Cooperative Perception**

## Files
- `index.html` — main page
- `style.css` — styles
- `assets/figure1.pdf` — main figure PDF (add this file)
- `assets/figure1.png` — fallback image placeholder, can be replaced
- `assets/video.mp4` — demo video (add this file)
- `assets/video_poster.png` — placeholder poster image
- `assets/CooperDrive_paper.pdf` — optional paper PDF for the Paper button

## What you should add
Place these files into the `assets/` folder:
- `figure1.pdf`
- `video.mp4`
- optionally `CooperDrive_paper.pdf`

## Deploy on GitHub Pages
1. Create a GitHub repository named `cooperdrive`
2. Upload all files in this folder
3. Go to **Settings → Pages**
4. Under **Build and deployment**, choose **Deploy from a branch**
5. Select branch `main` and folder `/ (root)`
6. Save and wait for deployment

Your page URL will be:

`https://YOUR-USERNAME.github.io/cooperdrive/`
